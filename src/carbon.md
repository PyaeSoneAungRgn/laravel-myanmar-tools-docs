# Illuminate\Support\Carbon

## Date

### Check Independence Day

```php
use Illuminate\Support\Carbon;

Carbon::parse('2023-01-04')->isIndependenceDay();

// true
```

### Check Union Day

```php
use Illuminate\Support\Carbon;

Carbon::parse('2023-02-12')->isUnionDay();

// true
```

### Check Peasants Day

```php
use Illuminate\Support\Carbon;

// true
```

### Check Labour Day

use Illuminate\Support\Carbon;
